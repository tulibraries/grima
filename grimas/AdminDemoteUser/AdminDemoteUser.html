AdminDemoteUser - remove admin privileges

If you are logged in as an admin, you can remove admin privileges from a user in your institution so they can no longer create accounts.

Input

  • Username: username for the account you are demoting

You can also delete the account so they no longer have access to grima.

You can list all users, including their admin status, as well.

API requirements

  • (none)
