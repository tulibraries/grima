AdminAdminUser - promote a user to admin

If you are logged in as an admin, you can promote a user account to admin status so that they can create user accounts for your institution.

Input

  • Username: username for the account you are promoting

If you change your mind, you can remove the admin privilege.

You can list all current users, including their admin status, as well.

API requirements

  • (none)
