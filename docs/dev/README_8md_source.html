2 =======
3 
4 Whisper into Alma's ear using APIs to speed up workflows. With grima,
5 you can use small web-based tools to do tasks like:
6 * View / Print a MARC record
7 * View bibs, holdings, items in a hierarchy
8 * Delete a bib and all of its inventory
9 * Create a boundwith
10 * Quickly add lots of similar items to a set or serial
11 * ... whatever else you want to write with the grima library!

13 ## Getting started
14
